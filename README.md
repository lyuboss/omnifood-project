# Omnifood Project

A responsive food delivery landing page built as a practice project while following Jonas Schmedtmann's "Build Responsive Real-World Websites with HTML and CSS" course on Udemy.

## About

Omnifood is a fictional AI-powered food subscription service. This project demonstrates modern HTML5 and CSS3 techniques for building professional, responsive websites.

## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Responsive Design

## What I'm Learning

- Building responsive layouts that work across all devices
- Modern CSS techniques including Flexbox and Grid
- Semantic HTML structure
- CSS positioning and the box model
