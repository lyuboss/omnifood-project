# Omnifood — Premium AI-Powered Food Subscription 🥘

[![Netlify Status](https://api.netlify.com/api/v1/badges/50867825-780b-487b-99f5-19e833f20f0f/deploy-status)](https://app.netlify.com/sites/omnifood-lyuboss/deploys)

A high-conversion, responsive landing page built with a focus on **Vanilla CSS/JS architecture** and professional UI/UX principles. This project demonstrates the ability to convert complex designs into high-performance, maintainable code without the use of heavy frameworks.

🚀 **[View Live Demo](https://omnifood-lyuboss.netlify.app/)**

---

## 🛠️ Tech Stack & Skills

* **HTML5:** Semantic markup for accessibility and SEO optimization.
* **Vanilla CSS3:** Advanced **CSS Grid** for complex 2D layouts and **Flexbox** for precise component alignment.
* **Vanilla JavaScript:** Custom DOM manipulation for mobile navigation toggles and smooth-scrolling behavior.
* **Deployment:** Automated CI/CD workflow via **Netlify**.

---

## 🚀 Key Features & Implementation

* **Desktop-First Responsive Design:** Developed for high-resolution displays first, using strategic `@media` breakpoints to reflow layouts seamlessly for tablets and mobile devices.
* **Zero-Framework Architecture:** Hand-coded from scratch to maximize performance and demonstrate deep mastery of native web fundamentals.
* **Component-Based Styling:** Implemented a modular CSS system for buttons, cards, and typography to ensure design consistency and scalability.
* **User Interface Logic:** Engineered a custom sticky navigation bar and interactive hover states to enhance the overall user experience (UX).
